A fully automated bot that tweets a new song every day with a YouTube link and a random fun fact!

## Features
- Runs daily (use Task Scheduler or cron)
- Random song + YouTube link
- Random music fun fact
- Clean, readable code

## Setup
1. Get Twitter API v2 keys from https://developer.twitter.com
2. Rename .env.example to .env and fill your keys
3. pip install -r requirements.txt
4. Run: python main.py
